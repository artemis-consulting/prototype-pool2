Drug Data

Having drug-related information and tools at your fingertips is a critical part of your overall health and well-being. The data provided by the FDA Drug Monitor are the latest and most relevant available data available from the FDA. Use this site along with your physician's advice to make informed decisions about the drugs you take.

This site helps cross connect information between manufacturers, drug labels, adverse events and enforcement information at an unprecedented level. Users can start with any of the data-sets and cross-connect to any of the other data-sets.

For more information regarding the data sources used on this website, visit www.fda.gov.

Drug Labels and more Drugs

This section allows users to browse drug names, their labels, and see adverse reactions for a selected drug, as well as any enforcement actions for a selected drug. Start browsing drugs.

Adverse reactions and more Adverse Events

This section allows users to browse adverse events, see drugs associated with these adverse events, and also drilldown by attributes like gender. Start browsing adverse events.

Enforcement and more Enforcements

Allows users to browse by State, and see the associated drugs and drug manufacturers. Links from Enforcement actions to related Manufacturer information. Start browsing enforcement actions.

Drug makers and more Manufacturers

This section allows users to browse drug manufacturers that are part of our source data, as either a manufacturer, or as a party in an enforcement action. Start browsing manufacturers.
